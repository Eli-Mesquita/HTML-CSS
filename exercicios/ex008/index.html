Principais formatações de texto

Negrito / Destaque

Nesse frase temos um termo em negrito usando a tag 'b'

Nassa frase temos um termo em destaque usando a tag 'strong'

Itálico / Ênfase

Nessa frase temos um termo em italico usando a tag i (wrap with abbreviation)

Nessa frase temos um termo com Ênfase usando a tag 'en'

Marca-texto

podemos criar o efeito Marca-texto usando a tag MARK

podemos também mudar a cor do efeito 'Marca-texto' usando o parametro style

Tamanho do texto

Estamos criando um texto grande e um texto pequeno nesse paragrafo

Texto deletado

podemos marcar um texto como excluido para indicar que ele deve ser lido mas não considerdo, usando a tag 'DEL'

inclusão de texto

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi incluído posteriormente, usando a tag 'ins'

existe também o sublinhado usando a tag 'u'

Texto sobrescrito

podemos usar a tag sup para escrever potenciação, por exemplo x20 + 3

Texto subescrito

podemos usar a tag 'sub'para escrever compostos quimicos, como H2O
